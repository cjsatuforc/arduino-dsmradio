# arduino-dsmradio
Arduino Library implementing the DSM Serial protocol to enable MLP4DSM or DX4e radio support.

## Minimum hardware requirements
- Arduino with 3.3v power
- mlp4dsm or dx4e/dx5e DSM Radio from Spektrum Controller
